在html中，为了避免冲突，使用“<” 与”&”符号需要进行转义：“<”用&lt;来代替，“&”用&amp;来代替；”&”字符在网址中也要进行转换，转换为&amp;

在md语法中，使用“<”与“&”符号，md会自动进行上述的转换，保证html文档的兼容性；
